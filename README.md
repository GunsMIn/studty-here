# studty-here
스터디를 구하는 사이트
## 개발환경
<br>

- **Java 11**
- **Build** : Gradle 7.6
- **Framework** : Springboot 2.7.7
- **CI & CD** : GitLab
- **Server** : AWS EC2
- **Deploy** : Docker
- **IDE** : IntelliJ
